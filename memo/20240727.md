## Memory Management Functions
- stdlib.h library includes memory management  functions
- malloc(bytes) : a pointer
- calloc(num_items, item_size) : a pointer
- realloc(ptr, bytes)
  - Resizes the memory
- free(ptr)
  - Release the block
